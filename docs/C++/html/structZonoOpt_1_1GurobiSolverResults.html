Detailed Description

Solver-native solution metadata produced by the Gurobi backend.

When an OptSolution is produced by a Gurobi solve, OptSolution::external_results points to an instance of this class so the caller can read Gurobi-specific status, node counts, gap, and dual bound. Inspect via dynamic_cast / isinstance.
